Projects

Amazon clone

The Amazon Clone project is an initiative to recreate the core structure and design of Amazon's e-commerce platform using only HTML and CSS. The primary objective is to practice web development skills and gain a deeper understanding of how to build complex, responsive web pages.

HTML: To create the structural layout of the web pages, including elements like headings, paragraphs, images, and links. CSS: To style the web pages, ensuring they are visually appealing and consistent with Amazon’s design. CSS is also used to implement responsive design principles.
